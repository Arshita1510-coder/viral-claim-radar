About This Project

Fact-Check Copilot for Social Feeds is an AI powered misinformation detection system designed to analyze viral claims circulating on social media platforms such as Instagram, Twitter, and WhatsApp.

The rapid spread of misinformation online can influence public opinion, create panic, and spread false narratives. Our system helps users quickly verify the credibility of viral claims using automated analysis and trusted information sources.

Problem We Address

Social media allows information to spread extremely fast, but not all of it is accurate. Many false claims go viral before they can be verified. This project focuses on identifying such misinformation and providing users with reliable explanations and credibility scores.

Our Solution

Fact-Check Copilot acts as a verification assistant. Users can enter a claim from social media, and the system analyzes it using AI-based logic and known misinformation patterns. The system then returns a credibility result such as Supported, Refuted, or Uncertain along with explanation, sources, and confidence score.

Key Features

🔎 AI Claim Analysis

Automatically analyzes viral claims using intelligent verification logic.

📊 Evidence Card

Displays result with confidence score, explanation, and supporting sources.

🌍 Social Media Focus

Designed to verify misinformation commonly spread on Instagram, Twitter, and WhatsApp.

🎤 Voice Claim Input

Users can speak a claim directly using voice recognition.

Technologies Used

This project is built using modern web technologies including HTML, CSS, JavaScript for the frontend and Python (Flask) for backend claim analysis. It demonstrates how AI-driven systems can help combat misinformation in digital environments.
