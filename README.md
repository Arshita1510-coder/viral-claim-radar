🛰 Viral Claim Radar
Fact-Check Copilot for Social Feeds

Viral Claim Radar is an AI-powered fact-checking web application that helps users verify viral claims circulating on social media platforms such as Instagram, Twitter, and WhatsApp.

The system analyzes claims from text or screenshots and classifies them as Supported, Refuted, or Uncertain, while providing a confidence score, explanation, and credible evidence sources.

🚨 Problem

Misinformation spreads rapidly across social media, making it difficult for users to determine the credibility of viral posts. Manual fact-checking is slow and often unreliable.

Viral Claim Radar aims to provide a fast and accessible AI-assisted verification tool that helps users instantly evaluate suspicious claims.

🎯 Key Features
🔎 Claim Analysis – Detects and evaluates claims from social media posts
📊 Stance Classification – Supported / Refuted / Uncertain
📈 Confidence Score – Credibility percentage for each claim
📚 Evidence Explanation – Displays sources and explanations
🖼 Screenshot Upload – Analyze claims from screenshots
🎤 Voice Input – Speak claims directly for verification
🌍 Multilingual Support – Supports English and Hindi
🎨 Interactive UI – Clean result cards and theme switching
⚙️ How It Works
User enters a claim or uploads a screenshot of a viral post
The system extracts and analyzes the claim
The claim is compared with verified information sources
The system returns:
Claim classification (Supported / Refuted / Uncertain)
Confidence score
Evidence explanation
Source references
